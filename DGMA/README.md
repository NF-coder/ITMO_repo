## О курсе
Курс посвещён различным приложениям функций многих переменных.\
Изучаются различные типы поверхностных и криволинейных интегралов, а также основы теории поля.\
Полезность курса для СППО сомнительна, но в целом довольно интересно и без особого напряга. Для общего развития самое то.
